# 2026-06-09 함수와 클래스

## 오늘 배운 내용

- 기본형은 기본값을 가지고 있고 참조형은 값을 안가지고 있다.
- 테스트코드의 중요성

## 실습 코드

```java
 Cleric(String name, int HP, int MP) {
    this.name = "아서스";
    this.HP = HP;
    this.MP = MP;
}

Cleric(String name, int HP) {
    this.name = name;
    this.HP = HP;
}

Cleric(String name) {
    this.name = name;
}
```

## 어려웠던 점

- 테스트코드 작성
- 중복되지 않는 생성자 생성

## 해결 방법

- Given, When, Then 식으로 구성
- 계속 시도해서 해결
