# 2026-07-06 문자열 조작

## 오늘 배운 내용

- length(): 문자열의 길이를 반환합니다.
- isEmpty(): 문자열의 길이가 0인지(비어있는지) 확인하여 boolean 값을 반환합니다.
- contains(CharSequence s): 특정 문자열이 포함되어 있는지 여부를 확인합니다.
- startsWith(String prefix) / endsWith(String suffix): 문자열이 특정 문자열로 시작하거나 끝나는지 확인합니다.
- indexOf(String str) / lastIndexOf(String str): 특정 문자나 문자열이 처음 또는 마지막으로 등장하는 인덱스(위치)를 반환합니다. (찾지 못하면 -1 반환)
- charAt(int index): 지정한 인덱스 위치에 있는 문자(char)를 가져옵니다.
- substring(int beginIndex, int endIndex): 문자열의 일부분을 추출합니다. (beginIndex부터 endIndex - 1까지)
- toLowerCase() / toUpperCase(): 문자열을 모두 소문자 또는 대문자로 변환합니다.
- trim(): 문자열의 앞뒤에 있는 공백을 제거합니다.
- replace(CharSequence target, CharSequence replacement): 특정 문자열을 다른 문자열로 모두 치환합니다.

## 기억할 것

- String의 불변성(Immutable): Java에서 String 객체는 한 번 생성되면 값을 변경할 수 없습니다. + 연산자를 사용해 문자열을 연결하면 기존 문자열이 변경되는 것이 아니라 새로운 String
  객체가 계속 생성되므로, 반복문 내에서 대량으로 연결할 경우 메모리와 성능에 악영향을 미칩니다.
- StringBuilder 활용: 문자열을 빈번하게 수정하거나 추가해야 할 때는 StringBuilder 클래스를 사용하는 것이 효율적입니다. 내부적인 버퍼를 사용하여 문자열을 동적으로 변경할 수 있습니다.
- append(): 문자열을 뒤에 추가합니다.
- toString(): 최종적으로 StringBuilder 객체를 일반 String 객체로 변환합니다.

## 실습 코드

```java
package com.survivalcoding;

public class Word {
    private String letters;

    public Word(String letters) {
        this.letters = letters;
    }

    // i번째 글자가 모음인지?
    public boolean isVowel(int i) {
        if (letters == null || i < 0 || i >= letters.length()) {
            return false;
        }
        String target = letters.substring(i, i + 1).toLowerCase();
        return "aeiou".contains(target);
    }

    //i번째 글자가 자음인지?
    public boolean isConsonant(int i) {
        if (letters == null || i < 0 || i >= letters.length()) {
            return false;
        }

        if (isVowel(i)) {
            return false;
        }

        String target = letters.substring(i, i + 1).toLowerCase();
        char ch = target.charAt(0);

        return ch >= 'a' && ch <= 'z';
    }
}



```
