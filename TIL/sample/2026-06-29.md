# 2026-06-29 제네릭과 열거형

## 오늘 배운 내용

### 1. 제네릭

- 제네릭은 클래스나 메서드를 만들 때 타입을 미리 고정하지 않고, 사용할 때 정할 수 있게 하는 문법이다.
- `<T>`처럼 임시 타입을 만들어두고, 실제 사용할 때 `String`, `Integer`, `Hero` 같은 타입을 지정한다.
- 한 번 타입을 정하면 그 타입에 맞는 값만 넣을 수 있다.
- 내 느낌으로는 “아직 어떤 타입이 들어올지 모르니까 미정 타입으로 만들어둘게. 대신 사용할 때 정한 타입에 맞게만 써라”에 가까웠다.

```java
class Box<T> {
    private T value;

    public void setValue(T value) {
        this.value = value;
    }

    public T getValue() {
        return value;
    }
}
```

```java
Box<String> box = new Box<>();

box.setValue("hello");

String value = box.getValue();
```

### 2. 열거형

- 열거형은 정해진 값만 사용할 수 있게 하는 문법이다.
- `enum` 안에 선언된 값만 사용할 수 있다.
- 문자열이나 숫자로 상수를 관리하면 오타가 나거나, 정해지지 않은 값이 들어갈 수 있다.
- `enum`을 사용하면 정해진 선택지만 사용할 수 있어서 더 안전하다.
- 내 느낌으로는 “상수를 문자열이나 숫자로 두면 잘못된 값도 들어갈 수 있으니, 차라리 enum 안에 있는 값만 써라”에 가까웠다.

```java
enum Job {
    WARRIOR,
    WIZARD,
    ARCHER
}
```

```java
Job job = Job.WIZARD;
```

## 기억할 것

- 제네릭은 타입을 나중에 정할 수 있게 해준다.
- 제네릭은 아무 값이나 넣는 것이 아니라, 정한 타입만 넣게 해준다.
- 열거형은 정해진 값 중에서만 선택하게 해준다.
- 문자열 상수보다 `enum`이 더 안전하다.

| 개념 | 정리 |
|---|---|
| 제네릭 | 사용할 타입을 나중에 정한다 |
| 열거형 | 정해진 선택지만 사용할 수 있게 한다 |

## 실습 코드

```java
package com.survivalcoding.day260629;

public class Main {
    public static void main(String[] args) {
        Box<String> nameBox = new Box<>();
        nameBox.setValue("홍길동");

        System.out.println(nameBox.getValue());

        Job job = Job.WIZARD;

        if (job == Job.WIZARD) {
            System.out.println("마법사입니다.");
        }
    }
}

class Box<T> {
    private T value;

    public void setValue(T value) {
        this.value = value;
    }

    public T getValue() {
        return value;
    }
}

enum Job {
    WARRIOR,
    WIZARD,
    ARCHER
}
```

## 어려웠던 점

- 제네릭의 `T`가 처음에는 실제 클래스 이름처럼 보여서 헷갈렸다.
- 하지만 `T`는 실제 타입이 아니라, 나중에 정할 타입을 임시로 표시한 것이라는 점을 알게 되었다.
- enum은 단순한 상수 모음처럼 보였지만, 정해진 값만 사용하게 강제하는 것이 중요했다.

## 해결 방법

- 제네릭은 “타입을 나중에 정하는 틀”이라고 생각했다.
- enum은 “정해진 선택지만 허용하는 상수 묶음”이라고 생각했다.
- 문자열로 직접 비교하기보다 enum 값을 비교하는 방식으로 이해했다.

```java
if (job == Job.WIZARD) {
    System.out.println("마법사입니다.");
}
```
