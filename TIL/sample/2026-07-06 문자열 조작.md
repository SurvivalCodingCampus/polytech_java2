# 2026-07-06 자바 학습 정리 : 문자열 조작

## 오늘 배운 내용

- 문자열 처리

## 기억할 것

| 메서드                         | 설명                                     | 사용 예시                   | 결과                     |
|:----------------------------|:---------------------------------------|:------------------------|:-----------------------|
| **`substring(start, end)`** | `start`부터 `end` **직전**까지 자르기           | `str.substring(0, 5)`   | `"Hello"`              |
| **`replace(old, new)`**     | 특정 문자열을 다른 문자열로 치환                     | `str.replace("l", "X")` | `"HeXXo, WorXd!"`      |
| **`split(regex)`**          | 구분자를 기준으로 문자열을 배열로 쪼개기                 | `str.split(",")`        | `["Hello", " World!"]` |
| **`toLowerCase()`**         | 모든 문자를 소문자로 변환                         | `str.toLowerCase()`     | `"hello, world!"`      |
| **`contains(str)`**         | 특정 문자열이 포함되어 있는지 여부 (`boolean`)        | `str.contains("Hello")` | `true`                 |
| **`startsWith(str)`**       | 특정 문자열로 시작하는지 여부 (`boolean`)           | `str.startsWith("He")`  | `true`                 |
| **`endsWith(str)`**         | 특정 문자열로 끝나는지 여부 (`boolean`)            | `str.endsWith("!")`     | `true`                 |
| **`indexOf(ch/str)`**       | 앞에서부터 검색하여 첫 번째로 등장하는 인덱스 반환           | `str.indexOf('o')`      | `4`                    |
| **`lastIndexOf(ch/str)`**   | 뒤에서부터 검색하여 첫 번째로 등장하는 인덱스 반환           | `str.lastIndexOf('o')`  | `8`                    |
| **`charAt(i)`**             | $i$번째 인덱스에 위치한 문자 딱 하나를 `char` 타입으로 반환 | `str.charAt(1)`         | `'e'`                  |

## 실습 코드 (테스트 구조 검증)

```Java
class WordTest {

    @Test
    @DisplayName("Word 클래스의 자음/모음 및 예외 처리 통합 테스트")
    void testword() {
        Word apple = new Word("Apple");

        // 대소문자 모음 검증
        assertTrue(apple.isVowel(0), "'A' -> 모음 (true)");
        assertTrue(apple.isVowel(4), "'e' -> 모음 (true)");
        assertFalse(apple.isVowel(1), "'p' -> 자음이므로 모음 아님 (false)");

        // 자음 검증
        assertTrue(apple.isConsonant(1), "'p' -> 자음 (true)");
        assertFalse(apple.isConsonant(0), "'A' -> 모음이므로 자음 아님 (false)");

        // 예외 처리
        Word mixed = new Word("a1!");

        // 숫자와 특수문자 검증
        assertFalse(mixed.isVowel(1));
        assertFalse(mixed.isConsonant(1));
        assertFalse(mixed.isVowel(2));
        assertFalse(mixed.isConsonant(2));

        // 0미만 인덱스
        assertFalse(mixed.isVowel(-1));
        assertFalse(mixed.isConsonant(-1));

        // 길이 초과 인덱스
        assertFalse(mixed.isVowel(100));
        assertFalse(mixed.isConsonant(100));

        // 문자 길이와 같은 인덱스
        assertFalse(mixed.isVowel(3));
        assertFalse(mixed.isConsonant(3));
    }
}
``````
