# 2026-07-06 정규표현식 / 문자열 처리

## 오늘 배운 내용
- 정규표현식 기본 문법
- split()으로 문자열 분리
- replaceAll()로 문자열 치환
- String.format()으로 문자열 포맷

## 기억할 것
- `.*` 모든 문자열, `\\d` 숫자, `[A-Z]` 대문자, `{n}` n개, `?` 0or1개
- `$` 는 문자열 끝을 의미 (끝 문자 제거할 때 유용)
- 정규표현식에서 `\`는 `\\\\`로 표현
- split()결과 배열은 끝 콤마 포함해도 정상 동작

## 어려웠던 점
- 정규표현식에서 \를 표현할 때 \\\\로 써야 하는 것

## 해결 방법
- 정규표현식 특수문자는 \\를 앞에 붙여서 이스케이프
