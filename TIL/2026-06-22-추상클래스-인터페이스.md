# 2026-06-22 추상클래스와 인터페이스

## 오늘 배운 내용

- 추상 클래스는 공통된 데이터와 기능을 모아두는 부모 클래스이다.
- 추상 클래스는 직접 객체를 생성할 수 없고 자식클래스들이 공통된 필드와 메서드를 상속받기 위해 사용한다. 
- 인터페이스는 반드시 구현해야 할 기능을 정의만 하고 실제 구현은 하지 않는다.

## 기억할 것

- 추상클래스는 객체 생성 불가능.
- 인터페이스의 메서드는 구현을 강제한다.
## 실습 코드

```java
 public interface Thing {
    double getWeight(); void setWeight(double weight); 
}
```

## 어려웠던 점

- 추상 클래스와 인터페이스가 둘 다 상속처럼 보여 차이를 이해하기 어려웠다.
- 일반 클래스와 추상 클래스의 차이가 거의 보이지 않았다
